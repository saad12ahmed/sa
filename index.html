My Projects

Speaker Diarization System

Speaker Diarization is the task of identifying the start and end time of a speaker in an audio file, together with the speaker's identity, i.e. who spoke when. It can enhance the readability of an automatic speech transcription by structuring the audio stream into speaker turns and providing the speaker's true identity. This project was implemented using a BI-LSTM network and Mean shift clustering algorithm.

Emotion Analysis in Text using TF-IDF

The process of identifying human emotion is known as emotion recognition. The underlying concept used here is LSA (Latent Semantic Analysis). It is an information retrieval technique that analyses the text in unstructured data and describes its relationship. It is used to analyze words based on the context used, followed by the Tf-IDF method, which assigns weights to terms based on Inverse Document Frequency (IDF). The output of this system is any of these six predefined emotion sets: Joy, sadness, anger, love, fear, and surprise.

Andoid chat Application

Yatter is easy to use Android application which is developed using Java. It uses Firebase to store the login information and the chats among the users of the application. Some of the features of this application are point-to-point communication and multicast group message communication from one sender to many receivers. Using this application, we can schedule messages to send them to the receiver at a particular time. The main objective of this project is to facilitate text messaging, group chatting options, data transfer without size restriction quickly and efficiently.

Pothole Detection System

Pothole Detection System is a sensor-based detection system that assists the driver in avoiding potholes on the roads by giving prior warnings. Warnings can be like a buzzer that is activated when the driver is approaching a pothole. The proposed method was implemented using Ultrasonic sensors and an accelerometer.
